Icon Component Size Test

Testing Width/Height Token Resolution

This test validates that the Icon component properly handles token keys and direct CSS values for width and height.

Width: '24px' (width token key)
Expected: 24px wide icon
Height: '6' (height token key)
Expected: 24px tall icon (height['6'] = '24px')
Width: '32px' (direct CSS value)
Expected: 32px wide icon
Height: 40 (number value)
Expected: 40px tall icon

Token Reference

Width Tokens (key: value)

  • '24px': '24px' - Standard icon size
  • '32px': '32px' - Large icons
  • '16px': '16px' - Small icons

Height Tokens (key: value)

  • '6': '24px' - Standard icon height
  • '8': '32px' - Large icon height
  • '4': '16px' - Small icon height
