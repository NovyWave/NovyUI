## Usage Examples

```typescript
// Import tokens
import { color, spacing, typography, width, height } from './tokens';

// Use reactive color tokens (automatically switches with theme)
const buttonStyle = {
  backgroundColor: color.primary['7'].value,
  color: color.neutral['1'].value,
  padding: `${spacing['12px']} ${spacing['16px']}`,
  fontSize: typography.size['16px'],
  fontWeight: typography.weight['6'],
  borderRadius: cornerRadius['4px'],
  width: width['256px'],
  height: height['10']
};
```

> All components and blocks reference these tokens. No hardcoded values are used in NovyUI components. Colors are reactive and automatically adapt to light/dark themes.
