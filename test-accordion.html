‚ö° Functionality Tests

Single Mode Behavior

  • Only one item can be open at a time
  • Opening an item closes the previously open item
  • Clicking an open item closes it
  • Default open items work correctly

Multiple Mode Behavior

  • Multiple items can be open simultaneously
  • Opening an item doesn't affect other items
  • Each item can be toggled independently
  • Default open items work correctly

Controlled vs Uncontrolled State

  • Uncontrolled mode works with defaultOpenItems
  • Controlled mode works with openItems prop
  • update:openItems event fires correctly
  • External controls can manage accordion state

Animation & Transitions

  • Expand/collapse animations are smooth
  • Icon rotation animation works correctly
  • No jarring or flickering during transitions
  • Animations respect reduced motion preferences

  • Headers are announced as buttons
  • Expanded/collapsed state is announced

  • Disabled items are announced as unavailable
  • Custom icons have appropriate labels

Color Contrast

  • Text meets WCAG AA contrast ratios
  • Focus indicators have sufficient contrast
  • Disabled state maintains readable contrast
  • Icons have adequate contrast
